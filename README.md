# Lane Assist

Lane Assist is a living, modular toolkit that encodes an opinionated definition of “what good looks like” at every stage of the product lifecycle through [Agent Skills](https://agentskills.io). Designed for innersource collaboration, it lets teams contribute, remix, and adopt proven approaches with confidence.

# Try in Claude Code, Claude.ai, and the API

## Claude Code

You can register this repository as a Claude Code Plugin marketplace by running the following command in Claude Code:

```
/plugin marketplace add jackrwstevenson/lane-assist
```

Then, directly install via:

```
/plugin install lane-assist@lane-assist
```

After installing the plugin, you can use the skill by just mentioning it.

## Claude.ai

To use any skill from this repository or upload custom skills, follow the instructions in [Using skills in Claude](https://support.claude.com/en/articles/12512180-using-skills-in-claude#h_a4222fa77b).

## Claude API

You can use Anthropic's pre-built skills, and upload custom skills, via the Claude API. See the [Skills API Quickstart](https://docs.claude.com/en/api/skills-guide#creating-a-skill) for more.
